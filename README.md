Smart Inventory Management System


A Smart Inventory Management System built using modern technologies to manage products, suppliers, inventory, sales, purchases, alerts, analytics, forecasting, and reports — all in one platform.

📌 Tech Stack
🔹 Backend
Node.js
Express.js
MongoDB
JWT Authentication + RBAC
🔹 Frontend
React
Axios (API calls)
Context API (State Management)
🔹 Other Tools
PDF & Excel Report Generation
Future Desktop Support via Electron
